# Шрифт Suisse Intl

Suisse Intl — коммерческий шрифт от [Swiss Typefaces](https://www.swisstypefaces.com/fonts/suisse). Для использования на сайте нужны веб-файлы (.woff2).

Поместите в эту папку файлы:
- `SuisseIntl-Regular.woff2` (400)
- `SuisseIntl-Medium.woff2` (500)
- `SuisseIntl-SemiBold.woff2` (600)

Можно экспортировать .woff2 из fonttools или Font Squirrel Webfont Generator из .ttf/.otf.
